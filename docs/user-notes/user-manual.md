## 平台概览

**ETF投研平台**是一款面向专业投资者和机构的全市场ETF数据投研工具。覆盖A股、港股、美股等市场的**1,500+只ETF**产品，提供实时行情、技术指标、综合评分、板块轮动、策略回测等核心投研能力。

### 核心功能一览

| 功能 | 说明 |
|------|------|
| **全市场ETF数据库** | 1,500+只ETF的基础信息、实时规模、分类归属、管理公司，每周自动扫描新增/退市 |
| **多维度综合评分** | 收益、风险、夏普、流动性、趋势五维度加权打分，全市场+分类内双重排名 |
| **智能筛选器** | 按市场、分类、RSI、夏普、波动率、收益率等20+条件组合筛选，支持常用预设 |
| **K线技术分析** | Candlestick K线图 + MA均线 + 布林带 + RSI14 + MACD副图，30/60/120/250日时间范围 |
| **板块轮动分析** | 多周期板块收益排名、相对强度(RS)、动量排名、轮动切换信号检测 |
| **策略回测引擎** | 动量/均值回归/RSI策略模板，历史回测 + NAV曲线 + 交易记录 + 绩效归因 |
| **标的池管理** | 自定义组合、权重配置（等权/评分/风险平价）、分类分布、相关性热力图、快照 |
| **报告与推送** | HTML/Markdown报告自动生成，企业微信/飞书/钉钉 Webhook 定时推送 |

## 快速入门（5分钟上手）

**Step 1 — 登录系统**

打开浏览器访问平台地址，进入登录页面。输入用户名和密码，点击「登录」按钮进入首页看板。

**Step 2 — 浏览首页，了解全局**

首页看板展示ETF总数、评分覆盖、分类数等统计卡片，以及**综合评分 Top 10**和**最近关注列表**。点击任意统计卡片可快速跳转到对应功能页。

**Step 3 — 筛选高评分ETF**

进入「全市场筛选器」页面 → 点击预设标签「高分高流动性」或手动设置"评分最小: 70" → 查看结果。表格每行可点击跳转到ETF详情。

**Step 4 — 查看K线和指标**

在详情页的「K线行情」Tab中查看蜡烛图，勾选 MA / 布林带 / RSI14 / MACD 复选框叠加技术指标。切换时间范围（30/60/120/250日）查看不同周期走势。

**Step 5 — 创建标的池**

进入「标的池管理」→ 点击「新建池」→ 输入名称 → 在详情页添加ETF成员 → 调整权重 → 查看分类分布饼图和相关性热力图。

**Step 6 — 运行策略回测**

进入「策略管理」选择模板 → 创建策略 → 进入「回测管理」点击「新建回测」→ 选择策略和ETF → 设定日期区间和初始资金 → 开始回测，查看净值曲线和绩效指标。

> **💡 提示：** 绝大多数页面的表格行、卡片、统计数字都是可点击的，点击即跳转到对应的详情页。这种"下钻"设计让你能快速从概览深入到细节。

### ETF详情

单只ETF的完整画像，三个Tab覆盖行情、指标、评分。

**📈 Tab 1 — K线行情**

- **蜡烛图** — 红涨绿跌，支持缩放拖拽，下方附成交量柱状图
- **时间范围** — 30日 / 60日 / 120日 / 250日，一键切换
- **技术指标叠加**（复选框控制显示/隐藏）：
  - **MA均线** — MA5（白）、MA10（黄）、MA20（紫）、MA60（蓝）
  - **布林带** — 上轨/中轨(MA20)/下轨，价格触及上下轨提示超买超卖
  - **RSI14 副图** — 0-100范围，水平虚线标注70超买线和30超卖线
  - **MACD 副图** — DIF快线 + DEA慢线 + 红绿柱状图，金叉死叉一目了然

**📊 Tab 2 — 指标统计**

以卡片形式展示该ETF最新的12个核心技术指标：

| 指标 | 说明 | 典型用途 |
|------|------|----------|
| RSI14 | 14日相对强弱 | >70超买、<30超卖 |
| 夏普比率(1Y) | 年化风险调整收益 | >1优秀、>0.5合格 |
| 波动率(20d) | 20日年化波动率 | 越低越稳定 |
| 最大回撤(1Y) | 一年内最大峰谷跌幅 | 越小抗跌性越强 |
| 1月 / 3月 / 1年 收益 | 不同持仓周期的回报率 | 评估短中长期表现 |
| MA5 / MA20 | 5日/20日均线值 | 判断短期/中期趋势方向 |
| ATR14 | 14日平均真实波幅 | 波动剧烈程度 |

**🎯 Tab 3 — 综合评分**

- **雷达图** — 五维度蛛网图（收益/风险/夏普/流动性/趋势），面积越大综合实力越强
- **综合评分** — 加权总分（0-100），配合颜色渐变进度条
- **全市场排名 / 分类排名** — 完全市场排名和同类别内的相对位置
- **各维度详细分数** — 五维度单独分数，方便定位强弱项

> **💡 提示：** K线图支持鼠标滚轮缩放和拖拽平移。双击图表可重置视图。指标复选框状态会保存在浏览器中，下次打开同页面自动恢复。

### 全市场筛选器

基于多维度技术指标的智能筛选工具，20+筛选条件自由组合。

**快速筛选预设**

页面顶部提供一键预设标签，点击即筛选：

- **高分高流动性** — 综合评分高 + 换手率充足
- **低波动稳健** — 波动率低 + 夏普比率高
- **强势趋势** — 趋势得分高 + 近期动量强
- **超跌反弹关注** — RSI偏低 + 存在均值回归可能

再次点击已选中的标签可以取消该预设（恢复该维度为不限）。

**高级筛选条件**

| 条件 | 说明 | 常用参考范围 |
|------|------|--------------|
| 市场 | SH(上海) / SZ(深圳) | — |
| 分类 | ETF行业/主题分类，下拉列表显示各分类ETF个数 | — |
| 评分 ≥ | 综合评分下限（0-100） | 60 ~ 80 |
| RSI ≥ | 14日RSI下限 | 30 ~ 50（避开超买区） |
| 夏普 ≥ | 1年夏普比率下限 | 0.5 ~ 1.0 |
| 波动率 ≤ | 20日年化波动率上限 | 0.2 ~ 0.3（稳健型） |

**结果表格与分页**

- 显示：代码、分类、评分、RSI、夏普、1月/3月/1年收益、波动率
- 支持每页 20 / 50 / 100 / 200 条切换，页码导航浏览全部结果
- 顶部显示"共 **X** 只ETF，第 Y/Z 页"
- 点击「重置条件」清空所有筛选条件

> **💡 提示：** 筛选条件支持组合使用，所有条件同时满足才显示结果。

### 评分排名

全市场ETF综合评分排行榜，支持按不同评分模板查看。

**评分模板切换**

顶部Tab栏可切换不同的评分维度组合：

- 支持按评分模板筛选和按市场/分类过滤

### 标的池管理

自定义ETF投资组合的全生命周期管理工具。

**池列表与创建**

- **创建池** — 点击「新建池」按钮，输入名称和描述
- **池列表** — 展示所有池，含成员数、创建时间等信息
- **删除池** — 确认后彻底移除

**池详情（三Tab布局）**

**Tab 1 — 成员与权重**

- 添加/移除ETF成员，查看成员列表
- **目标权重** — 使用滑块或输入框调整各成员的权重占比（自动归一化至100%）
- **建议权重** — 一键生成三种算法权重方案：
  - **等权** — 所有成员相同权重
  - **评分加权** — 按综合评分高低分配权重
  - **风险平价** — 按波动率倒数分配，低波高配

**Tab 2 — 持仓分布**

- **分类饼图** — 按分类维度展示池内ETF的分布（支持数量和权重两种模式）
- **池绩效统计** — 组合整体1月/3月收益、夏普比率、最大回撤

**Tab 3 — 相关性热力图**

- 池内ETF两两价格相关性矩阵
- 颜色从深绿（正相关+1.0）到深红（负相关-1.0）
- 帮助发现过度集中风险（高相关性）或构建分散化组合

**快照功能**

创建池快照可保存某个时间点的池配置和权重，用于后续对比分析。在「快照记录」区查看历史快照列表。

## 投研分析

### 相关性分析

分析多只ETF之间的价格相关性，辅助构建低相关性的分散化投资组合。

**操作步骤**

1. 在ETF选择器中搜索并添加ETF（**2~20只**）
2. 选择计算窗口期：`30日` / `60日` / `120日` / `250日`
3. 选择相关系数方法：**Pearson**（线性相关）或 **Spearman**（秩相关，更稳健）
4. 查看热力图 — 每格显示相关系数，颜色从绿（正相关）到红（负相关）渐变

提供快捷添加按钮：宽基指数组 / 科技组 / 消费组 / 医药组，一键添加常用ETF组合。

> **💡 使用场景：** 如果你想构建一个分散化的组合，应选择相关性较低的ETF（热力图中颜色偏红/浅色的配对）。如果两只ETF相关性 > 0.85，说明它们高度同涨同跌，同时持有没有分散效果。

---

### 收益对比

多只ETF的历史收益曲线叠加对比，直观比较不同标的表现。

**操作步骤**

1. 选择多只ETF进行对比（**2~10只**）
2. 选择时间范围：`1月` / `3月` / `6月` / `1年` / `全部`
3. 选择显示模式：
   - **归一化收益** — 将所有ETF的起点设为单位净值1，对比相对涨幅
   - **每日百分比** — 展示每日涨跌幅的百分比变化
4. 曲线图中每条线代表一只ETF，颜色图例标注在下方

同样提供快捷添加按钮，方便快速选中常用对比组。

### 板块轮动

分析不同ETF分类（板块/行业）之间的轮动节奏，识别市场热点切换。

**核心指标解读**

| 指标 | 公式/口径 | 解读 |
|------|-----------|------|
| 板块收益 | 该分类下ETF在时间窗口内的平均收益率 | 单位：% |
| 相对强度 (RS) | 板块收益率 ÷ 市场平均收益率 | RS > 1 表示跑赢市场 |
| 动量排名 | 按过去N周收益排序（1为最佳） | 排名上升 = 动能增强 |
| 夏普比率 | （板块年化收益 - 无风险利率）÷ 年化波动率 | 风险调整后收益质量 |
| 轮动信号 | 检测领先板块在两个周期之间的切换 | 上一期领先→本期落后的切换 |

**页面布局**

- **顶部统计栏** — 分析日期、市场平均1月收益、板块数量
- **轮动信号警告** — 当检测到领先板块切换时显示提醒
- **收益率柱状图** — 各板块1月收益率排名（正收益红色，负收益绿色）
- **相对强度柱状图** — 各板块RS值，虚线标注 RS=1.0 基准线
- **详细数据表** — 动量排名、收益率、夏普、波动率、RSI、相对强度

> **💡 使用场景：** 在月末或季末查看板块轮动，了解资金在不同行业间的流向变化。关注排名稳步上升的板块（动能积聚），警惕前期领先板块开始走弱（可能反转）。

## 策略系统

### 策略管理

管理量化策略配置，每个策略基于一个预设模板，可自定义关键参数。

**3种预设策略模板**

| 模板 | 核心逻辑 | 关键参数 |
|------|----------|----------|
| **动量策略** | 价格趋势追踪：N日收益超过阈值则买入，低于负阈值则卖出 | momentum_window (20-60), threshold (0.03=3%) |
| **均值回归** | 价格偏离均值后倾向于回归：Z-score低于下限买入，高于上限卖出 | mean_reversion_window (20-60), z_score_threshold (1.5-2.5) |
| **RSI策略** | 基于超买超卖：RSI低于超卖区买入，高于超买区卖出 | rsi_period (14), overbought (70), oversold (30) |

**策略生命周期**

1. **创建** — 选择模板 → 命名策略 → 调整参数 → 保存

### 绩效归因

对回测结果进行收益来源拆解，理解策略超额收益的来源。

**Brinson归因模型**

将策略超额收益分解为三部分：

- **配置效应（选股）** — 因持有不同于基准ETF权重而产生的收益差异
- **择时效应** — 因买卖时机不同于基准而产生的收益差异
- **交互效应** — 配置和择时交叉影响的部分 = 总超额 - 配置 - 择时

页面展示归因瀑布图和收益来源饼图，帮助理解策略的收益来源结构。

**访问方式**

在回测详情页点击「绩效归因」按钮，或在策略/回测关联菜单中进入归因分析页。基准默认为等权ETF市场组合。

## 附录

### 技术指标详解

#### MA — 移动平均线 (Moving Average)

计算指定周期内收盘价的算术平均值。MA5 = 近5日均价（周线），MA20 = 近20日均价（月线），MA60 = 近60日均价（季线）。

**常用信号：** 短期均线上穿长期均线 = **金叉（看涨）**；短期均线下穿长期均线 = **死叉（看跌）**。

#### RSI — 相对强弱指标 (Relative Strength Index, 14日)

衡量价格变动的速度和幅度，取值0-100。由 Wilder 于1978年提出。

**解读：** RSI > 70 = 超买区域（可能回调）；RSI < 30 = 超卖区域（可能反弹）。注意：强势趋势中RSI可在超买区持续较长时间。

#### MACD — 指数平滑异同移动平均线

由 DIF（快线，EMA12-EMA26）、DEA（慢线，DIF的9日EMA）和 MACD柱（DIF-DEA，×2）组成。

**信号：** DIF上穿DEA = **金叉**；DIF下穿DEA = **死叉**；MACD柱由负转正 = 多头动能增强；顶背离/底背离 = 趋势反转预警。

#### 布林带 — Bollinger Bands

中轨 = 20日均线；上轨 = 中轨 + 2×标准差；下轨 = 中轨 - 2×标准差。覆盖约95%的价格波动区间。

**信号：** 价格触及上轨 = 短期偏高；触及下轨 = 短期偏低；带宽收窄 = 波动率收缩，可能预示突破；带宽扩大 = 波动率放大，趋势行情。

#### ATR — 平均真实波幅 (Average True Range)

衡量价格波动幅度（非方向），取真实波幅的N日移动平均。ATR越高，日内或日间波动越剧烈。

**用途：** 设定止损位（如2×ATR）、评估交易标的的波动风险、仓位管理参考。

#### 夏普比率 — Sharpe Ratio

公式：(Rp - Rf) / σp，其中Rp = 年化收益率，Rf = 无风险利率，σp = 年化波动率。衡量单位风险获得的超额回报。

**评价标准：** > 1.0 = 优秀；0.5 ~ 1.0 = 良好；< 0.5 = 一般。注意：负夏普表示收益低于无风险利率。

#### 波动率 — Volatility (20日年化)

日收益率的标准差 × √252。衡量价格相对于均值的偏离程度，是资产风险的核心度量。

低波动率（< 20%）适合稳健型投资者；高波动率（> 40%）潜在收益大但回撤风险也大。

#### 最大回撤 — Max Drawdown

从历史最高净值点到之后最低点的累计跌幅百分比。衡量最坏情况下的潜在亏损。

回撤 < 10% = 优秀风控；10%-20% = 正常范围；> 30% = 风险较高。关注回撤修复时间（回到前高所需天数）。

### 常见问题

**Q: 数据更新频率是多少？**

A: 日K行情和技术指标每日收盘后更新（约18:00-20:00），综合评分和板块轮动在指标更新后自动计算。交易信号每日09:00自动生成。ETF基础信息每周日03:00扫描更新。

**Q: 为什么收益率是红涨绿跌？**

A: 平台遵循中国A股市场惯例：**红色表示上涨/正收益，绿色表示下跌/负收益**。这与中国投资者的视觉习惯一致，与欧美市场颜色惯例相反。

**Q: 综合评分是怎么计算的？**

A: 综合评分基于五个维度：收益能力、风险控制、夏普比率、流动性、趋势强度。每个维度先做全市场百分位排名（0-100），再按评分模板的权重加权求和。不同模板（均衡型/收益优先/稳健型）的权重分配不同。最终得分是相对排名分，而非绝对指标值。

**Q: 为什么我只看到50条数据？怎么看到全部ETF？**

A: 全市场筛选器和ETF列表都支持分页。顶部状态栏显示"共 X 只ETF"，使用分页控件可以浏览全部数据。可在右下角切换每页显示条数：20/50/100/200。

**Q: 如何创建自定义标的池？**

A: 进入「标的池管理」→ 点击「新建池」→ 输入名称和描述 → 进入详情页 → 点击「添加成员」→ 输入ETF代码（如 510300.SH）→ 设置权重 → 保存。也可以使用「建议权重」一键生成等权/评分加权/风险平价方案。

**Q: 回测结果可信吗？**

A: 回测基于历史数据模拟，存在**过拟合、幸存者偏差、未来函数、交易成本未计入**等固有限制。回测表现好的策略在实盘中可能显著走弱。建议：①样本外验证 ②使用多ETF交叉验证 ③关注夏普比率和最大回撤而非仅有总收益 ④不要仅凭回测结果做出投资决策。

**Q: 平台支持哪些ETF市场？**

A: 目前主要覆盖中国A股市场ETF（上海证券交易所 + 深圳证券交易所），总计1,500+只。港股、美股ETF数据可通过yfinance数据源获取，加密货币通过OKX/CCXT获取。

**Q: 布林带和RSI同时显示，K线图会不会很乱？**

A: 不会。所有技术指标通过复选框独立控制，你可以只勾选需要查看的指标。MA均线叠加在主图上；布林带也用半透明色叠加在主图；RSI和MACD有独立的副图区域。推荐同时开启MA20 + 布林带 + MACD，关闭其他以保持图面清晰。

**Q: 如何获取各平台机器人的Webhook地址？**

A: ①企业微信：群设置 → 群机器人 → 添加 → 复制Webhook地址。②飞书：群设置 → 群机器人 → 添加自定义机器人 → 复制Webhook地址。③钉钉：群设置 → 智能群助手 → 添加机器人 → 自定义 → 复制Webhook地址（可选择加签验证）。

**Q: 忘记密码怎么办？**

A: 目前平台使用简单的用户名/密码认证。如忘记密码，请联系系统管理员在数据库中重置。后续版本将增加密码找回功能。
