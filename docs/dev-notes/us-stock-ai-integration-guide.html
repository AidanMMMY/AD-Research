5.3 AI 功能指南

A. 研究笔记生成

# API方式生成研报
curl -X POST http://localhost:8000/api/v1/research/notes/generate \
  -H "Authorization: Bearer " \
  -H "Content-Type: application/json" \
  -d '{"instrument_code": "SPY.US"}'

查询研报

curl http://localhost:8000/api/v1/research/notes/SPY.US \
  -H "Authorization: Bearer "
生成逻辑: 1. 拉取近30天日线、最新指标(RSI/MA/MACD/波动率)、综合评分 2. 构建数据化Prompt → LLM生成2-3段中文研报 3. 解析JSON输出 (content/summary/sentiment/confidence) 4. 存储到 research_note 表

B. 情绪分析

# 触发新闻抓取+情绪分类
curl -X POST "http://localhost:8000/api/v1/research/sentiment/AAPL.US/ingest?days=3" \
  -H "Authorization: Bearer "

查询情绪聚合

curl "http://localhost:8000/api/v1/research/sentiment/AAPL.US?days=7" \
  -H "Authorization: Bearer "
数据流: Finnhub新闻 → LLM逐条分类(positive/negative/neutral + score) → 入库 → 聚合统计

C. AI对话助手

# 创建会话
curl -X POST http://localhost:8000/api/v1/research/chat/sessions \
  -H "Authorization: Bearer " \
  -H "Content-Type: application/json"

发送消息

curl -X POST http://localhost:8000/api/v1/research/chat/sessions/1/messages \
  -H "Authorization: Bearer " \
  -H "Content-Type: application/json" \
  -d '{"content": "SPY最近表现怎么样？技术面怎么看？"}'
数据感知机制: 1. 用户消息中正则提取标的代码 (如 SPY.US) 2. 自动查询最新指标 + 评分数据 3. 作为System Prompt附件注入LLM上下文 4. AI回答引用的真实数据而非幻觉

9. 常见问题

Q: yfinance报429/无数据？

A: yfinance是第三方库，Yahoo随时可能限流或改HTML。系统已内置三重降级链： 1. Tiingo (需 TIINGO_API_KEY, 1000次/天) 2. Finnhub (需 FINNHUB_API_KEY, 60次/分) 3. FMP (需 FMP_API_KEY, 250次/天)

Q: LLM API费用会超预算吗？

A: 系统默认使用Claude Haiku ($0.25/MTok)，加上Redis缓存，月度$5-15即可。可在 app/config.py 设 llm_monthly_budget 控制上限。如果未设 ANTHROPIC_API_KEY，AI功能会自动跳过（不会crash）。

Q: S&P 500个股发现只跑了一次？

A: 调度器每周日02:00自动运行。手动运行： 

python -c "
from app.core.database import SessionLocal
from app.data.pipelines.us_stock_discovery import USStockDiscoveryPipeline
p = USStockDiscoveryPipeline(SessionLocal())
p.run_with_retry()
"

Q: 颜色惯例切换后所有页面都生效了吗？

A: 以下组件已支持： - ✅ KLineChart (蜡烛图颜色、成交量、MACD) - ✅ ReturnTag (收益率标签) - ✅ Dashboard, ScoreRanking, PoolDetail (通过ReturnTag) - ❌ CorrelationHeatmap (待优化，热力图颜色通常与市场方向无关)

Q: AI对话如何获取数据？

A: ChatService会自动正则匹配消息中的代码（如 AAPL.US），查询最新指标和评分，作为System Prompt附件。AI回答会引用真实数据而非生成幻觉。
