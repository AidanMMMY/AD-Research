1. Robinhood

定位: 美国最大零售券商，0 佣金，移动优先，面向新手与进阶投资者。

1.1 信息架构与首屏

  • 首屏全屏 Hero 视频/动画 + 单一 CTA（"Get started" Sign up）
  • 首屏后纵向滚动展示产品：Invest → Options → Crypto → Learn → 24/7 Support
  • 导航: Sticky top bar，左侧 Logo，右侧 Login / Sign up
  • 底部 Footer 包含法律披露

来源: https://robinhood.com/us/en/

1.2 色彩系统

TokenHex用途
主背景（dark）#110E08默认深色背景
纯黑#000000数字主题区块
霓虹绿强调#CCFF00CTA 按钮、品牌色
白色#FFFFFF面板卡片
二级暖色#1C180D浅色背景变体

默认深色模式，App 内 Settings → Appearance 可切 light。涨跌: 绿涨红跌。

来源: https://robinhood.com/us/en/ CSS 提取

1.3 字体系统

  • 正文: Capsule Sans Text（自研几何无衬线）
  • 标题: Nib Pro Display / Martina Plantijn（衬线标题）
  • 代码/数据: Capsule Sans Text Mono
  • 后备: Geist（Vercel 开源）

1.6 可借鉴模式

  • 极简首屏 CTA：一屏一事，Hero 只放品牌价值主张 + 一个 CTA
  • 自研字体品牌感
  • 深色默认 + Light 可选
  • 数据卡片大字号关键数字（~32px+）

2. Webull 微牛

定位: 中美跨境零售券商，注重技术分析工具，面向活跃交易者。

2.2 色彩系统

TokenHex用途
主背景#FFFFFF白色底
文本主色#2B3240深灰蓝
次要文本rgba(0,0,0,.45)说明文字
Modal 圆角12px弹窗
Button 圆角6px按钮

2.3 字体系统

  • 拉丁: OpenSansV2（Light–ExtraBold + Italic）
  • CJK: NotoSans（Thin–Black 9 档）、HarmonyOSSans
  • 衬底标题: DM Serif Display
  • 备用: InterWB

来源: https://www.webull.com CSS 提取

2.5 可借鉴模式

  • 多语言字体栈: OpenSans + NotoSans + HarmonyOS
  • 三层阴影层级系统
  • 社区 Feed + 行情数据同屏

3. Interactive Brokers

定位: 全球最大电子经纪商，服务专业交易员/机构。

  • TWS: 多窗口 Mosaic 模块化布局，学习曲线陡峭
  • Client Portal: Web 版简化界面，响应式设计
  • 数据密度极高，色彩保守（蓝色强调），绿涨红跌

可借鉴

  • 三档信息密度: Compact / Comfortable / Spacious
  • 模块化 Dashboard 可拖拽

4. TradingView

定位: 全球最大图表分析与交易社区。

  • 核心: Canvas/WebGL 图表引擎，占首屏 70%+
  • K 线涨: #26A69A / 跌: #EF5350
  • 暗色背景: #131722，网格: #2A2E39
  • 内置 Color Blind 预设 —— 极少数色盲友好图表工具
  • Pine Script 指标生态，charting_library 可嵌入

5. 富途牛牛 Futu

定位: 香港第一投资平台，覆盖港股/美股/A 股。

  • 六端全覆盖: iOS/Android/macOS/Windows/Linux/HarmonyOS
  • AI 走势预测 + 专业画线工具
  • 多市场颜色自适应: A 股红涨绿跌 vs 港股绿涨红跌
  • 牛牛圈: 实盘社区 + 行情同屏

来源: https://www.futunn.com SSR 页面确认

6. Tiger Brokers 老虎证券

  • 移动优先，深色主题为主
  • 社区"老虎社区" + 模拟交易内置

7. Public.com

  • 社交 Feed 为核心，替代传统行情列表
  • 教育嵌入: 每个资产页内置 Why/How 模块
  • 碎片投资: 小数点份额降低心理门槛

8. Seeking Alpha

  • 品牌橙 #ff7200，深色横幅 #414a5f
  • 阶梯付费墙: Free → Premium → Pro
  • 量化评级: Strong Buy → Strong Sell 五档可视化

来源: https://seekingalpha.com HTML 源码

9. FinChat

  • Astro 框架（Vercel checkpoint 确认）
  • 对话式 UI: 自然语言 → 结构化数据 + 交互式图表
  • SaaS 定价: Free / Plus $29/mo / Pro $79/mo

10. AD-Research 设计建议清单

P0 — 必须实现（MVP 阻塞项）

P0-1 深色模式默认 + 亮色可选，light-dark() CSS 函数统一 token → Robinhood, TradingView
P0-2 涨跌颜色四重冗余：颜色 + ▲▼ 箭头 + +/− 符号 + 文字标签 → TradingView 色盲模式
P0-3 K 线涨跌用蓝 #0072B2 + 橙 #E69F00（Okabe-Ito），替代传统红绿
P0-4 三档信息密度：Compact / Comfortable / Spacious → IBKR TWS
P0-5 prefers-reduced-motion 适配：reduce 时 0ms 动效 → WCAG 2.2
P0-6 自选列表实时价格更新 + motion-safe: 颜色闪烁 → Webull, 富途

P1 — 应该实现（v1.0 质量目标）

P1-1 模块化 Dashboard 可拖拽面板 → IBKR Mosaic
P1-2 图表嵌入 Widget（复用 TradingView charting_library）
P1-3 移动端底部 Tab Bar（自选/行情/研究/AI/设置）→ Robinhood, 富途
P1-4 数据卡片大号关键数字 + Sparkline → Robinhood
P1-5 筛选器面板多条件筛选 → Webull, TradingView
P1-6 色盲模式 Toggle → TradingView
P1-7 多市场颜色自适应 → 富途牛牛
P1-8 响应式断点 mobile-first → Robinhood
P1-9 社区/讨论集成 → Public.com, 富途牛牛圈

P2 — 建议实现（v1.5+ 差异化）

P2-1 AI 对话式投研界面 → FinChat
P2-2 量化评级系统五档可视化 → Seeking Alpha
P2-3 分析师/策略 Marketplace → Seeking Alpha
P2-4 教育嵌入模块 → Public.com
P2-5 自定义字体品牌感 → Robinhood Capsule Sans
P2-6 Pine Script 式自定义指标 → TradingView
